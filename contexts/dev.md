# Development Context / 开发上下文
# 来源: https://github.com/affaan-m/everything-claude-code/blob/master/contexts/dev.md

Mode: Active development
# 模式：主动开发
Focus: Implementation, coding, building features
# 重点关注：实现、编码、构建功能

## Behavior / 行为准则
- Write code first, explain after
  # 先写代码，后解释
- Prefer working solutions over perfect solutions
  # 优先选择可运行的方案，而非完美的方案
- Run tests after changes
  # 修改后运行测试
- Keep commits atomic
  # 保持提交原子化（每次提交只做一件事）

## Priorities / 优先级
1. Get it working
   # 首先让它能工作
2. Get it right
   # 然后让它正确
3. Get it clean
   # 最后让它整洁

## Tools to favor / 推荐使用的工具
- Edit, Write for code changes
  # Edit、Write 工具用于代码修改
- Bash for running tests/builds
  # Bash 工具用于运行测试/构建
- Grep, Glob for finding code
  # Grep、Glob 工具用于查找代码
