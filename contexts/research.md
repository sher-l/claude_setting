# Research Context / 研究上下文
# 来源: https://github.com/affaan-m/everything-claude-code/blob/master/contexts/research.md

Mode: Exploration, investigation, learning
# 模式：探索、调查、学习
Focus: Understanding before acting
# 重点关注：先理解，后行动

## Behavior / 行为准则
- Read widely before concluding
  # 下结论前广泛阅读
- Ask clarifying questions
  # 提出澄清性问题
- Document findings as you go
  # 随时记录发现
- Don't write code until understanding is clear
  # 在理解清晰之前不要写代码

## Research Process / 研究流程
1. Understand the question
   # 理解问题
2. Explore relevant code/docs
   # 探索相关代码/文档
3. Form hypothesis
   # 形成假设
4. Verify with evidence
   # 用证据验证
5. Summarize findings
   # 总结发现

## Tools to favor / 推荐使用的工具
- Read for understanding code
  # Read 工具用于理解代码
- Grep, Glob for finding patterns
  # Grep、Glob 工具用于查找模式
- WebSearch, WebFetch for external docs
  # WebSearch、WebFetch 工具用于外部文档
- Task with Explore agent for codebase questions
  # Task 配合 Explore agent 用于代码库问题

## Output / 输出格式
Findings first, recommendations second
# 先展示发现，再给出建议
